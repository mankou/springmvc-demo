【工程说明】

  1. springmvc-demo   父工程 用于maven中聚合项目 方便将多个项目执行一个命令就能打包发布等。
  2. springmvc-demo-common  common工程 放一些公用的处理类
  3. springmvc-demo-server  一般用于开发自己业务逻辑的工程
  4. springmvc-demo-web   web工程，其是一个空web工程，什么逻辑都没有 就是为了能在tomcat中运行而已，其依赖server工程


【其它目录说明】
bin 该工程用到的脚本

resource  有该工程需要用到的一些私有的jar包 和数据库创建表的脚本



【配置文件说明】

web.xml web相关的配置 
/springmvc-demo-web/src/main/webapp/WEB-INF/web.xml


spring mvc相关的配置
/springmvc-demo-web/src/main/webapp/WEB-INF/config/mvc-dispatcher-servlet.xml


spring 相关的配置
包括集成hibernate、事务配置等
/springmvc-demo-web/src/main/webapp/WEB-INF/config/applicationContext-local.xml


log4j配置
/springmvc-demo-web/src/main/webapp/WEB-INF/config/log4j.xml



【重要类说明】

统一返回
GlobalRestfulResponseBodyAdvice
该类统一处理spring mvc的返回值，其返回的类的格式为 JsonResult


统一异常处理
GlobalExceptionHandler
该类统一处理异常
