简要：该工程是一个基于maven的 spring mvc demo工程，其是一个web工程，其整合了springmvc、 hibernate、log4j 
具有如下功能：可接收REST请求，能操作数据库、基于注解、统一返回、统一异常处理。对于初学者可用于熟悉如何搭建工程，对于成手可将该工程快速应用于自己的项目


相关软件说明
JDK 1.7
eclipse 4.6
maven 3.3.3
oracle 11G
spring 4.3.8.RELEASE
hibernate 4.3.10.Final
log4j 1.2.17
